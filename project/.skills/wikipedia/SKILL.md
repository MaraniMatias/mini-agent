Use the WebSearch tool to look up information on Wikipedia.

- Always search in the language most likely to return useful results (prefer English for broad topics).
- After receiving the summary, extract the key facts relevant to the user's question.
- If the summary is insufficient, search again with a more specific or alternative query.
- Cite the source URL at the end of your answer.
- Never fabricate facts — only use what the Wikipedia summary provides.
